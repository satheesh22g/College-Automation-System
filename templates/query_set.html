Find student information, attendance, marks, and academic data

How to Use This Search

This tool allows you to search for student information using various methods:

  • By Roll Number: Enter a roll number like "161CS027"
  • By Name: Search for a student by their full or partial name
  • By Command: Use special commands to view specific data

Results will display matching student profiles with their academic information, attendance records, and marks if available.

Example Queries
  • show my attendance - View your attendance records
  • show my marks - Check your academic marks and grades
  • 161CS027 - Search by roll number
  • John Doe - Search by student name
